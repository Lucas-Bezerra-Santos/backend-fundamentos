# Fase 12 — Projeto Integrador Final (4+ semanas)

**Objetivo:** Consolidar TUDO construindo um sistema real.

## Projeto: Sistema de E-commerce Simplificado

### Componentes

| Componente | Fundamentos exercitados |
|------------|------------------------|
| API Gateway | Load balancing, rate limiting, auth |
| Auth Service | JWT, OAuth, hashing, sessions |
| Product Catalog | SQL modeling, indexing, pagination |
| Order Service | Transactions, ACID, saga pattern |
| Payment Service | Idempotência, retry, circuit breaker |
| Notification Service | Message broker, filas, async processing |
| Search Service | NoSQL, caching, denormalization |
| Monitoring | Logs, metrics, health checks |

### Requisitos

- [ ] Usar Clean Architecture ou Hexagonal
- [ ] Implementar pelo menos 2 serviços que se comunicam via message broker
- [ ] Usar banco relacional + cache (Redis)
- [ ] Implementar auth com JWT
- [ ] Testes unitários + integração
- [ ] Docker Compose para rodar tudo local
- [ ] Documentar decisões de arquitetura (ADRs)

### Checklist por Serviço

- [ ] Auth Service
- [ ] Product Catalog
- [ ] Order Service
- [ ] Payment Service
- [ ] Notification Service
- [ ] Search Service
- [ ] Monitoring setup
