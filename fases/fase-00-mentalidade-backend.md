# Fase 0 — Mentalidade Backend (1 semana)

**Objetivo:** Entender como o pensamento backend difere do frontend.

No frontend você pensa em: UI, estado local, renderização, UX.
No backend você pensa em: **dados, concorrência, durabilidade, consistência, segurança e escala.**
